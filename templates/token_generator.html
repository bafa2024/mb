Frequently Asked Questions

  1. Go to Mapbox Tokens
  2. Create a new token
  3. In the Secret scopes section, check tokens:write
  4. Create the token and use it as your master token here

Mapbox doesn't allow modifying scopes of existing tokens for security reasons. You must create a new token with the required scopes. This tool helps you do that programmatically.

  • Public scopes: Basic permissions like styles:read, fonts:read
  • Secret scopes: Advanced permissions like tilesets:write, tokens:write
The MTS (Mapbox Tiling Service) requires secret scopes, which are only available for secret tokens (starting with sk.).
